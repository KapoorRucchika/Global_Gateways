# 🌍 Global Gateways

Global Gateways is a travel website built using Node.js, Express, and EJS. It provides users with an engaging platform to explore popular destinations and travel packages with a clean and responsive UI.

---

## 🚀 Features

- ✨ Responsive homepage with Bootstrap carousel  
- 🌎 Popular destinations section  
- 💡 "Why Choose Us" feature highlights  
- 🔁 Reusable EJS components (Header & Footer)  
- 📦 Routing for travel packages  
- 🎨 Clean UI with Bootstrap & Font Awesome  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, Font Awesome  
- **Backend:** Node.js, Express.js  
- **Templating Engine:** EJS  
- **Version Control:** Git & GitHub
